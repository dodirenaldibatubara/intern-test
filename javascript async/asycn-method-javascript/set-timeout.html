Belajar javascript Asyncronus

Method async yang tersedia pada javascript salah satunya ialah set timeout

artinya di sini ialah kita membuat program yang bukan syncronus yaitu porgram di buat tidak mengikuti urutan default. pada contoh di bawah kita akan membuat elemen baru pada javascript, dan element itu akan di eksekusi pada saat yang kita butuhkan, pada kasus ini kita set waktu sebanyak 5000 ms, artinya kode akan di eksekusi 5 detik kemudian, setelah dia di jalankan.
